Morphology

Let’s:

  1. code a regular morphology for the emergent language.
  2. look at using it as a part of the inductive bias for the emergent language.
  3. an algorithm for an agent to associate the morphology with the states (for senders perhaps) what is the minimal steps to identify the morphology to the states. (a form of grounding)

A simple generative model for morphology

import random

random.seed(45)
# start with a simple morphology and then add more complexity.

vowels = ['a','e', 'i','o', 'u', 'aa','ee', 'ii','oo', 'uu','ai','au','ei','ou','ia','ua']
consonants = ['b', 'c','cs','ch', 'd','dh', 'dzh','f', 'g','gh', 'h','hw','ny', 'j', 'k','kw' 'l','ld','lh', 'm','mb', 'n','nc', 'nd','ng','ngw','nqu','nqt','nt', 'p', 'q', 'r','rd','rh', 's', 'sh', 't','tsh' 'v', 'w', 'x', 'y', 'z']

def generate_rnd_phone():
    # generate a random phoneme
    return random.choice(consonants) + random.choice(vowels)

print(f'{generate_rnd_phone()}')

def generate_num_phoneme(consonant, vowel):
    # pick a consonants cons from consonants
    c = consonants[consonant % len(consonants)]
    # pick a vowel from vowels
    v = vowels[vowel % len(vowels)]
    return c + v 
    
print(f'{generate_num_phoneme(3, 2)}')

# next let generate a random root word with k phonemes

def generate_rnd_stem(k=3):
    # generate a random word with k phonemes
    word = ''
    for i in range(k):
        word += generate_rnd_phone()
    return word 

print(f'{generate_rnd_stem(3)}')
